# 🛒 E-Commerce Storefront

A modern and responsive E-Commerce Storefront built using React.js, Redux Toolkit, React Router, and FakeStore API. The application provides a complete shopping experience including product browsing, filtering, sorting, wishlist management, cart functionality, checkout process, dark mode, and responsive design.

---

## 🚀 Features

### Product Management

* Product Catalog
* Product Search
* Category Filters
* Price Filters
* Product Sorting

  * Price Low → High
  * Price High → Low
  * A → Z
  * Z → A

### Shopping Experience

* Product Details Page
* Shopping Cart
* Wishlist
* Recently Viewed Products
* Checkout Page
* Order Summary

### User Interface

* Responsive Design (Mobile, Tablet, Desktop)
* Professional Hero Section
* Professional Footer
* Dark Mode Toggle
* Back To Top Button
* Custom 404 Not Found Page

### Performance Optimization

* Redux Toolkit State Management
* React.memo Optimization
* useMemo Optimization
* React.lazy & Suspense
* Code Splitting
* Lazy Loading

### Additional Features

* Static Customer Reviews & Ratings
* Social Media Links in Footer
* Mobile Hamburger Navigation
