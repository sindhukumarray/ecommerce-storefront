# 🛒 E-Commerce Storefront

A modern E-Commerce Storefront built using React, Redux Toolkit and FakeStore API.

---

## 🚀 Features

- Product Catalog
- Product Search
- Category Filters
- Price Filters
- Product Details
- Shopping Cart
- Wishlist
- Recently Viewed Products
- Checkout Page
- Order Summary
- Redux Toolkit State Management
- React.memo Optimization
- useMemo Optimization
- React.lazy & Suspense
- Responsive UI

---

## 🛠️ Technologies Used

- React.js
- Redux Toolkit
- React Router DOM
- FakeStore API
- CSS3
- JavaScript (ES6)
